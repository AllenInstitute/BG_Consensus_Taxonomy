Documented in this directory is the Human and Macaque data pull for the BICAN/HMBA BasalGanglia publication pacakges.

## Code

`data_pull`: Contains bulk downloading scripts which should be general enough for re-use. Included is the metadata tracker containing critical information about google buckets and libraries to gather.

`NeMO_data_handling.py`: Merges anndata file and adjusts meta.data for each species according to both Lydia Ng's requirements and AIT schema.
